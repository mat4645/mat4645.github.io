Privacy Policy

This defines how user data is handled at ConvertFileBox (hereinafter referred to as "this site").

1. Security of Input Data (Local Processing Principle)

In the conversion tools provided by this site, all data (CSV, JSON, XML, text, etc.) input or uploaded by the user is processed entirely on the user's web browser (local environment).

This data is never sent, stored, or recorded on our servers or any third-party servers. Even in an offline state where you are not connected to the internet, the conversion features will continue to work once the page has been loaded.
