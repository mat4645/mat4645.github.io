オンラインツール一覧

全てのツールはブラウザ内で完結し、データがサーバーに送信されることはありません。

CSV → JSON 変換 (File型)

ファイルを選択して読み込みます。Shift-JIS等の文字コードも自動判別して正確に変換可能です。

CSV → JSON 変換 (貼り付け型)

CSVテキストを直接貼り付けて変換します。クイックな確認や、少量のデータ変換に最適です。

JSON → CSV 変換

JSONデータをExcel等で扱いやすいフラットなCSV形式に変換します。ネストした構造にも対応。

JSON 形整器 (Formatter)

崩れたJSONを綺麗に整形します。Minify（軽量化）機能や構文チェック機能も搭載。

CSV 文字コード変換 (Enc)

UTF-8からShift-JIS、あるいはその逆など、CSVの文字コードのみを安全に変換・ダウンロードします。

JSON → JSON Schema 生成

JSONデータの構造を解析し、 Draft-07 準拠の JSON Schema を自動的に生成します。API開発に便利。
